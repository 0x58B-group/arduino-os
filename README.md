# arduino-os
tiny single-task os for Arduino.

# Hardware
A button on pin D2
B button on pin D3
(All buttons are HIGH-enabled)
DHT11 sensor on pin D4
Beeper on pin D5
HC-SR04	Trig pin on D6
		Echo pin on D7
Rotate sensor on A0
IIC LCD1602 on IIC pins, addr=0x27(default), you can change in the source code.
